# raw/ 운영 규칙

## 역할
불변 원본 소스 저장소. AI는 읽기만 한다. 절대 수정하지 않는다.

## 하위 폴더
| 폴더 | 저장할 것 |
|------|-----------|
| `articles/` | 웹 아티클, 블로그 포스트 (Obsidian Web Clipper로 클리핑) |
| `books/` | 책 챕터 노트, 독서 메모 |
| `papers/` | 논문 PDF 또는 마크다운 변환본 |
| `lectures/` | 강의 노트, 강의 자료, 유튜브 강의 메모 |
| `projects/` | AX 프로젝트 관련 문서, 회의록, 기획안 |
| `conversations/` | 중요한 대화 메모, 인터뷰 내용 |
| `assets/` | 이미지 파일 (Obsidian 첨부파일 폴더로 설정 권장) |

## 파일 네이밍
- `YYYY-MM-DD_제목.md` 형식 권장
- 예: `2026-06-15_RAG_시스템_설계_패턴.md`

## Obsidian 설정 권장
- Attachment folder path → `raw/assets/`
- Web Clipper 저장 경로 → 각 하위 폴더로 지정
