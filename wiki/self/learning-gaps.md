## 위키 전체 공백 — 아직 페이지가 없는 것들

> 인제스트된 소스에서 언급됐지만 아직 개념 페이지가 없는 항목들.

- [ ] **GraphRAG** — VectorDB + GraphDB 결합 검색. 여러 페이지에서 언급됐지만 페이지 없음.
- [ ] **RAG 파이프라인** — 청킹·임베딩·검색·생성 전 단계를 다루는 개념 페이지
- [ ] **임베딩 / VectorDB** — 온톨로지와 대비되는 검색 방식으로 자주 언급
- [ ] **KIE (Key Information Extraction)** — 기업-문서-자동화에서 언급, 실무 경험 있음

## 운영 규칙

- 새 소스 인제스트 후: 새로운 `빈 곳` 항목을 이 파일에 추가
- 공백이 채워진 경우: 해당 항목을 `~~취소선~~`으로 표시 + 날짜 기재
- 새 개념 페이지 생성 시: "위키 전체 공백" 항목에서 해당 항목 제거
