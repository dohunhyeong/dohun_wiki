# wiki/ 운영 규칙

## 역할
AI가 전적으로 작성하고 유지하는 지식 위키. Dohun은 읽고 방향을 잡는다.

## 하위 폴더
| 폴더 | 저장할 것 |
|------|-----------|
| `concepts/` | AI, AX, 통계, 기술 개념 페이지 |
| `projects/` | AX 프로젝트별 위키 (문제, 접근법, 결과, 교훈) |
| `self/` | Dohun 자신에 대한 지식 (결정 패턴, 회고, 강점/약점) |
| `people/` | 협업자, 고객사, 주요 인물 페이지 |

## 페이지 생성 기준
- 새 소스 인제스트 시 → 요약 페이지 생성
- 3회 이상 언급된 개념 → 개념 페이지 생성
- 가치 있는 쿼리 답변 → 새 페이지로 저장
- 모든 페이지는 `[[관련 페이지]]` 링크 포함

## 필수 파일
- `index.md` — 전체 위키 목록. 질문 시 가장 먼저 참조.
- `log.md` — append-only 작업 이력.
