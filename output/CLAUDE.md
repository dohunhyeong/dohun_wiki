# output/ 운영 규칙

## 역할
wiki/ 의 지식으로부터 생성된 결과물 저장소.

## 하위 폴더
| 폴더 | 저장할 것 |
|------|-----------|
| `reports/` | 분석 리포트, 조사 결과 문서 |
| `presentations/` | 발표 자료 (Marp 형식 권장) |
| `memos/` | 초안, 메모, 짧은 글 |

## 생성 원칙
- 항상 wiki/ 페이지를 출처로 명시
- 파일명: `YYYY-MM-DD_제목.md`
- 결과물이 새로운 인사이트를 담고 있으면 wiki/ 에도 페이지 추가
